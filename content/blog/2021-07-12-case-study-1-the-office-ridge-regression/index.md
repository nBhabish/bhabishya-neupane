# Fit ridge regression

``` r
theoffice_model <- linear_reg(mixture = 0, penalty = 0.1) %>% 
  set_engine("glmnet")
theoffice_model
```

## Creating workflow

``` r
theoffice_wf <- 
  workflow() %>% 
  add_recipe(theoffice_recipe) %>% 
  add_model(theoffice_model)
```

## Fit workflow

``` r
theoffice_ridge <- 
  theoffice_wf %>% 
  fit(data = theoffice_train)
theoffice_ridge %>% 
  yardstick::tidy()
```
